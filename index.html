¿Qué es esto?

Un generador simple de frases de contraseña que usa el método Diceware para generar contraseñas seguras.

¿Por qué recoge los movimientos de mi mouse?

La aplicación usa la librería seedrandom.js para reforzar el generador de números pseudo-random alimentándolo con coordenadas del cursor y otras fuentes de entropía. Si usas un navegador moderno, usará el (mucho más seguro) método window.crypto como fuente de números random.

¿Quieres tener uno propio?

El código completo de la aplicación está disponible en GitHub listo para que hagas con él lo que quieras.
